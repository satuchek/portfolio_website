Pass On, Ghostie

Pass On, Ghostie was a game that I created with a team during the 2018 Global Game Jam. We decided to create a game about a ghost that must possess others' bodies, and die in them, in order to interact with the physical world. I was responsible for implementing the systems that allow the ghost to possess bodies as well as the way the gravestones interact with the world.
